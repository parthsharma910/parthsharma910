# 💫 About Me:
Hi, I’m Parth Sharma, a B.Tech student specializing in Artificial Intelligence and Machine Learning (AIML) at Dronacharya College of Engineering. I have a strong interest in AI, machine learning, data structures, and problem solving. I enjoy learning new technologies and building projects that help me improve my coding skills. Along with software, I also have hands-on knowledge of hardware and embedded systems, working with Raspberry Pi, ESP32, and Arduino IDE. Currently, I am focusing on Python, C programming, and core concepts of data structures and machine learning. My goal is to become a skilled AI/ML engineer and work on innovative real-world solutions.<br>
